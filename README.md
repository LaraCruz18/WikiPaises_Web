WikiPaíses
Projeto front-end desenvolvido para a disciplina de Desenvolvimento de Plataformas Web. A aplicação consome a REST Countries API para listar países na tela inicial e exibir informações detalhadas em uma página separada.

Tecnologias
React com Vite

React Router DOM (navegação)

Axios (requisições HTTP)

CSS (estilização)
